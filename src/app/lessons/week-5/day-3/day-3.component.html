Setup for Routing

For those who were not in class on Day 20:

  • Clone this project.
  • Create a new branch called understanding-nav-paths.
  • Pull pratima branch from Github into understanding-nav-paths branch locally.
  • Create a new repository in your Github account and set that to be the new remote for the cloned project.
  • Push master to new repository.
  • Checkout to understanding-nav-paths (no need to push yet).
