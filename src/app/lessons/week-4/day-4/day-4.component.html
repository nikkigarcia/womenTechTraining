Practicing Nested Loops

Practice 1:

  • Create a bools property that is of type array of boolean values.
  • Assign true, false as values of bools array.
  • Create a nums property that is of type array of numeric values.
  • Assign 1, 2.5, 5 as values of nums array.
  • Create a strs property that is of type array of string values.
  • Assign hi, there as values of strs array.
  • Create a test property that is of type object.
  • Assign id: 1, name: 'test' as value of test object.
  • Create a user property that is of type object.
  • Assign id: 2, name: 'user' as value of user object.
  • Create a objs property that is of type array of object values.
  • Assign test and user properties as values of objs array.
  • Create a types property that is of type array of any values.
  • Assign bools, nums, strs and objs properties as values of objs array.
  • Create a getTypeof method that takes in an argument and returns typeof of the argument.

Practice 2:

  • Continue working on above project.
  • Create a new branch (branch off of previous branch).
  • In the template, display "Types" in a header element.
  • Create an un-ordered list under the header with a li tag.
  • Loop through types (call variable type & capture the index).
  • Inside above li, add a paragraph that has text-transform: capitalize style applied.
  • This paragraph should display the typeof of type variable (use getTypeof method).
  • Along with the paragraph, add an un-ordered list (inside the above li) which loops through each element in the type variable (call variable val.
  • Inside above li, do a ternary check.
  • The ternary condition: typeof of val is 'object'.
  • If this condition is true, display id and name of val variable.
  • Else display val variable.
